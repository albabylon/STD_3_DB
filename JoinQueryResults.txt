Кроме того что вы можете объединить таблицы и получать данные из объединённой таблицы, 
команда SELECT позволяет выполнять операции с выборками (результатами SELECT) как с множествами, а именно:

Объединение двух множеств — UNION. Результатом объединения двух множеств является множество, которое содержит в себе все элементы из первого и из второго множества.

Пересечение двух множеств — INTERSECT. В результирующее множество попадают те элементы, которые есть и в первом, и во втором множестве.

Последняя операция исключение — EXCEPT. В результирующее множество попадают те элементы, которые есть только в первом множестве.


select id from orders union select id from orders where status = 'done';
select id from orders union all select id from orders where status = 'done';
select id from orders except select id from orders where status = 'done';
select id from orders intersect select id from orders where status = 'done';

Для того чтобы эти операции сработали, необходимо, чтобы запросы возвращали одинаковое число столбцов. Типы данных у столбцов также должны совпадать.

______________________________________________________________
UNION
Строка включается в итоговое множество (выборку), если она присутствует хотя бы в одной из выборок. 
При этом дубликаты в результирующее множество не включаются. Для их включения нужно использовать UNION ALL.

С помощью скобок можно задавать порядок объединения. 
Это похоже на скобки в математическом выражении: вы сначала считаете выражения в скобках, а потом подсчитываете всё остальное выражение. 
В результирующем наборе используются имена столбцов, которые заданы в первом запросе. 
Вы можете использовать сортировку финальной выборки с помощью ORDER BY. 
Сортировка применяется к результату соединения, поэтому ORDER BY указывается только в конце всего составного запроса.


______________________________________________________________
UNION
Строка включается в итоговое множество, если она присутствует в каждом из них. 
Строки-дубликаты в результирующее множество не включаются. Для их включения нужно использовать INTERSECT ALL



______________________________________________________________
EXCEPT
Строка включается в итоговое множество (выборку), если она присутствует в первом множестве (выборке), но отсутствует во втором. 
Строки-дубликаты в результирующее множество не включаются. Для их включения нужно использовать EXCEPT ALL.
